Enables OmniFaces in Quarkus applications

== How to try it?

Just add the following dependency:

[source, xml]
----
<dependency>
    <groupId>com.github.adminfaces</groupId>
    <artifactId>quarkus-omnifaces</artifactId>
    <version>1.2.0</version>
</dependency>
----

=== Run sample application

* `cd system-tests && mvn compile quarkus:dev`
* access http://localhost:8080/index.xhtml

TIP: Run via quarkus runner with `mvn clean package && java -jar target/quarkus-app/quarkus-run.jar`

=== Native mode

To run o quarkus native use:

`mvn clean package -Pnative && ./target/quarkus-app/quarkus-run`

=== Run System tests

`mvn clean test`

=== Run native tests

`mvn verify -Pnative`
